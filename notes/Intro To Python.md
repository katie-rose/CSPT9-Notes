## Arithmetic Operators

Operators are special symbols that represent computations like addition and multiplication.

```
+	addition
-	subtraction
*	multiplication
/	division
//	floor division
**	exponentiation
%	modulus
```

```
>>> 30 + 6
36
>>> 40 - 4
36
>>> 6 * 6
36
>>> 72 / 2
36.0
>>> 6**2
36
>>> 36 % 35
1
>>>
```
